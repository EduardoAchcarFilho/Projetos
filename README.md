Cadastro de Tarefas/Erros, Ajustes e Preferências (TaskEdx)

App criado para melhora de desempenho e ter uma base de dados sobre a prestação de suporte técnico que eu realizava.

![Capturar](https://github.com/user-attachments/assets/fdf29f9f-d976-4136-884c-ba5576b2b761)

O objetivo final deste projeto era que, com esse monitoramento constante, eu teria uma visão geral e contínua do desempenho desses processos, possibilitando uma tomada de decisão rápida, economizando tempo e eliminando a necessidade de buscar dados dispersos em várias fontes, sempre aprendendo mais sobre o meu trabalho.
